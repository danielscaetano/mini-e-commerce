# Instalacao
primeiro instale o composer, php e npm, recomendo o https://laravel.com/docs/12.x/installation

# Inicializar projeto
use o comando no git bash ou no terminal do linux, cd suapasta/mini-e-commerce, e em seguida use o comando Composer setup, isso configura iniciara o banco o node e outras coisas

use o comando composer dev para inicar o projeto

vá no local host indicado, provavelmente http://127.0.0.1:8000

# Usando o projeto
clique em criar categoria, digite o nome da categoria desejada, e clique em criar a categoria quando desejar

para criar produto, vá em criar produto e escolha o nome do produto, uma categoria que você criou anteriormente, um valor, se achar necessário uma descrição, depois volte a tela home, se quiser pode clicar em mini-e-commerce no canto superior esquerdo

na tela home selecione os produtos que quiser e a quantidade de cada, depois coloque o nome do cliente no campo acima das escolhas de produtos, e clique em adcionar ao carrinho

quando terminar clique em pagar para finalizar a tranzação
